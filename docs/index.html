유저스크립트 매니저 설치

유저스크립트를 실행해주는 확장 프로그램이에요. 둘 중 아무거나 하나만 설치하면 됩니다. (이 증폭기는 둘 다에서 똑같이 동작해요)

Tampermonkey 설치 (추천) Violentmonkey (대안)

🐳 Whale(네이버)도 위 링크로 설치돼요. (설정에서 "다른 스토어 확장 설치 허용"이 필요할 수 있어요)

2

⚠️ 사용자 스크립트 허용 켜기 (꼭 필요!)

최신 크롬/웨일에서는 이걸 안 켜면 스크립트가 아예 실행되지 않아요. 아래 둘 중 하나만 하면 됩니다. (한 번만 하면 계속 유지돼요)

방법 A — 최신 크롬/엣지 (138+)
  1. Tampermonkey 아이콘 우클릭 → "확장 프로그램 관리"
  2. 상세 페이지에서 "사용자 스크립트 허용" 토글을 켜기
— 또는 (위 토글이 안 보이면) —
방법 B — 웨일 / 구버전 크롬
  1. 주소창에 chrome://extensions (웨일은 whale://extensions) 입력
  2. 우측 상단 "개발자 모드" 토글 켜기

✅ 켠 뒤에는 브라우저를 완전히 종료했다 다시 켜 주세요. (창만 닫지 말고 완전 종료)
